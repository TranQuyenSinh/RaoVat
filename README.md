# Giới thiệu
Website cho phép người dùng đăng tin mua bán, trao đổi sản phẩm cũ hoặc mới với nhiều danh mục sản phẩm.

# Công nghệ
- Server: ASP.NET Web APIs
- Database: MS SQL Server
- Frontend: ReactJS, Jquery, Bootstrap, SCSS
